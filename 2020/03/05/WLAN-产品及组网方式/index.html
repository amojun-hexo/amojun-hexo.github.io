[学习笔记]WLAN_产品及拓扑结构

发表于 2020-03-05 更新于 2020-03-06

这里主要介绍华为产品

产品的命名规则

PoE

  1. 概念
    • 全称： Power Over Ethernet，以太网供电
    • 是指通过10Base-T、100Base-TX、1000Base-T以太网网络进行供电，最大传输距离为100m
  2. PoE的目的
    • 节省电源布线成本
    • 结合UPS（不间断电源）提高可用性
    • 方便统一管理
  3. PoE术语
    • 协议的标准
      • IEEE 802.3af PoE标准 //最大输出功率为15.4W
      • IEEE 802.3at-2009 PoE 标准 //最大输出功率30W
    • PSE—电源源设备—供电设备
    • PSU—电源支持单元—电源
    • PD—被供电设备—受电设备
    • PI—电源接口—供电接口
  4. PoE组件

802.11基本元素

  1. BSS //基本服务集—由一组相互通信的工作站（STA）构成。
    • Independent BSS—独立BSS—没有AP参与连接，是全部由STA组成的无线网络。
    • Infrastructure BSS—基础结构BSS—有AP参与连接，STA连接到AP构成的无线网络。
  2. BSA //基本服务区域—BSS所覆盖的范围，即一个无线单元。
  3. ESS //扩展服务集—是指通过利用骨干网络将几个BSS串联在一起，形成的更大的服务集。
    • ESS使用的SSID相同
  4. SSID //服务集标识符—用于表示一个无线网络，并与其他无线网络进行区分。
    • 一个AP支持多个SSID。
  5. BSSID //基本服务集标识符—实际上就是AP的MAC地址，用来标识ESS中不同AP管理的BSS。
    • 华为推荐构成ESS的两个BSA之间重叠范围大小为15%～25%。

拓扑结构

AD-Hoc组网拓扑
基础架构组网拓扑
WDS组网拓扑
Mesh组网拓扑

AD-Hoc组网拓扑

  1. 无AP参与
  2. STA之间自行组网
  3. 临时性组网

基础架构组网拓扑

  1. 需要AP参与，STA与AP相连
  2. 分布式系统(DS)是接入点间转发帧的骨干网路
  3. 骨干网络属于有线的分布式系统（DS）
    • 如果仅为一个BSS，则没有什么特别要求
    • 如果利用骨干网络形成ESS，需要注意如下几点：
      • AP工作时使用的信道不能重叠
      • ESS中两个BSA的重叠区域范围推荐为10%～15%。

WDS组网拓扑

  1. 概念
    • WDS(无线分布式系统)：通过无线链路连接两个或者多个独立的有线局域网或者无线局域网，组建一个互通的网络，实现数据访问
  2. 需要AP参与，STA与AP相连
  3. 其优点在于组网的灵活性和便捷性
  4. 在部署时，网络结构分为
    • P2P
      Root AP:根AP Leaf AP:叶子AP
    • P2MP 
      Root AP:根AP Leaf AP:叶子AP
    • 手拉手
      Root AP:根AP Leaf AP:叶子AP middle AP:中间节点
适合室内环境
对带宽要求不高的用户可以用
    • 背靠背
      Root AP:根AP Leaf AP:叶子AP
      适合室外环境
对带宽要求高的用户可以用

Mesh组网拓扑

  1. 概念

    • 无线Mesh网络WMN是指利用无线链路将多个AP连接起来，并最终通过一个或两个Portl节点接入有线网络的一种星形动态自配置的无线网络

  2. Mesh组网中，AP是两两互联，形成全互连或部分全互连。

  3. Mesh组网中，AP有3种角色：

    • MPP—连接有线网络、MP的AP设备。
    • MP—用于连接MPP、MAP或其他MP的AP设备。
    • MAP—用于连接STA和MP的AP设备。
